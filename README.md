# Front-End Task TEMG4950L by Monica

This is a Cryptocurrency Dashboard with features as specified in the task requirement. The technology used in this project is JavaScript with React. For the components, this task used Material UI and Apex Chart. For the Crypto graph, the data has been processed first and is organized in new data files (included in the project folder). To ensure the information is persistent, localStorage is used.

## Link to Deployed Version

https://cryptodashboardyeay.onrender.com

## Starting the Project

In the project directory, you can run:

### `npm start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

The page will reload when you make changes.\
